    Steg 1: Home Assistant

    Börja här. När Home Assistant svarar kan Hemnav hämta Deltaco/Helo-lampor och kontakter som visas som light eller switch.

    API ej kontrollerat Kör lokalt API hemma för PC, PS5 och Samsung TV.
    1. Installera eller öppna Home Assistant hemma.
    2. Lägg Deltaco/Helo i Smart Life/Tuya eller Deltaco-appen.
    3. Koppla Tuya i Home Assistant.
    4. Skapa token i Home Assistant-profilen och klistra in den här.
    5. Tryck Testa Home Assistant, sedan Hämta enheter.
